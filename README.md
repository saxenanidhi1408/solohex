# solohex
Ai based system that detects fake news on social media by analyzing content and providing real time feedback to users.
Fake News Detector
Overview
This is a web application built using Flask that detects whether a news article is real or fake. Users can input news text and get a prediction with a short explanation. The project is designed for learning purposes and demonstrates basic NLP and web development concepts.
Features
Input any news text and get a prediction: Real or Fake.
Displays a short message explaining the result.
Simple and user-friendly interface.
Easy to run locally on your system.
Technologies Used
Python 3
Flask (for web framework)
HTML, CSS (for front-end)
Pickle (for loading trained model)
NLP model (trained to classify news as real or fake)
